Acerca de este proyecto

Este proyecto fue creado para la clase de Introducción a la Programación en Internet de la Universidad Autónoma de Baja California.

El objetivo de este proyecto es crear un sitio web utilizando los conocimientos adquiridos dentro del curso. La aplicación construida es una galeria que muestra una pequeña parte de los hobbies del desarrollador.

Este sitio web fue creado utilizando HTML, CSS y JavaScript.
